### Installation 🚀

Clon the repository
```
git clone https://github.com/Rafael3994/Shopping-List.git
```
Go the project folder and install the dependencies.
```
cd Shopping-List

## npm:
npm install 
```
Start the project.
```
npm run dev
```
```
http://localhost:4200/
```
